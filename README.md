# gif-creator
gif creation helper made for the CookieRun Wiki

# Options:
  -h, --help           show help message and exit
  -l, --label LABEL    Label of the type of gif
  -fps FPS             framerate of the gif from plist '3 numbers of precision required'
  -o, --output OUTPUT  Output name

# Requirements

Requires python and imagemagick to be on your computer and PATH
